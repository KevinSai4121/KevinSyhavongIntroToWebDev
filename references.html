References & Learning Resources

References are formatted in APA 7th style, with brief comments describing their relevance to this project.

References

  • Berners-Lee, T., & Fischetti, M. (1999). Weaving the Web. HarperCollins. – Provides foundational background on the creation of the World Wide Web and early Web 1.0 concepts.
  • Berners-Lee, T. (n.d.). The World Wide Web project. CERN. https://info.cern.ch/ – Offers a view of the first website ever created; useful for understanding early static Web 1.0 design.
  • World Wide Web Consortium. (n.d.). A little history of the World Wide Web. https://www.w3.org/History.html – Outlines major historical milestones in the evolution from Web 1.0 to Web 2.0.
  • World Wide Web Consortium. (n.d.). Semantic Web. https://www.w3.org/standards/semanticweb/ – Explains Semantic Web principles that inform core ideas behind Web 3.0.
  • Mozilla. (n.d.). AJAX. MDN Web Docs. https://developer.mozilla.org/en-US/docs/Web/AJAX – Used to describe how asynchronous requests enabled more dynamic Web 2.0 applications.
  • Fielding, R. T. (2000). Architectural styles and the design of network-based software architectures (Doctoral dissertation, University of California, Irvine). – Primary source on REST architecture, foundational to Web 2.0 and modern Web API design.
  • Ethereum Foundation. (n.d.). What is a smart contract? https://ethereum.org/ – Defines smart contracts and explains their role in Web 3.0 decentralized ecosystems.
  • Buterin, V. (2014). A next-generation smart contract and decentralized application platform. Ethereum Whitepaper. – Introduces concepts central to Web 3.0 such as dApps, decentralization, and programmable blockchain logic.

Good Learning Resources

  • Mozilla. (n.d.). HTML elements reference. MDN Web Docs. https://developer.mozilla.org/en-US/docs/Web/HTML/Element – Used to verify correct structure and semantics of HTML while building website pages.
  • Mozilla. (n.d.). CSS reference. MDN Web Docs. https://developer.mozilla.org/en-US/docs/Web/CSS/Reference – Used to look up CSS properties while designing responsive layouts and styling.
  • Mozilla. (n.d.). Web APIs. MDN Web Docs. https://developer.mozilla.org/en-US/docs/Web/API – Helpful for understanding browser APIs and JavaScript functionality relevant to interactive web content.
  • World Wide Web Consortium. (n.d.). Web architecture. https://www.w3.org/standards/webarch/ – Provides high-level design principles that guided the organization and structure of this project.
