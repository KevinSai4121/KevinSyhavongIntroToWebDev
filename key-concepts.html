Key Concepts

  1. URI/URL: Identifiers/locators for web resources.
  2. HTTP: Stateless application protocol for data exchange.
  3. HTML/CSS/JS: Content, presentation, and behavior of the web.
  4. AJAX: Asynchronous JS and XML/JSON for dynamic UIs.
  5. APIs (REST/JSON): Machine interfaces powering modern apps.
  6. Semantic Web: Linked data enabling machine-readable meaning.
  7. Decentralization: Reducing reliance on central intermediaries.
