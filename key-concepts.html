Key Concepts

  1. URI/URL: Identifiers/locators for web resources.
  2. HTTP: Stateless application protocol for data exchange.
  3. HTML/CSS/JS: Content, presentation, and behavior of the web.
  4. AJAX: Asynchronous JS and XML/JSON for dynamic UIs.
  5. APIs (REST/JSON): Machine interfaces powering modern apps.
  6. Semantic Web: Linked data enabling machine-readable meaning.
  7. Decentralization: Reducing reliance on central intermediaries.
  8. Protocol: A set of rules that define how data is formatted and transmitted between systems (for example, HTTP on the web or blockchain network protocols).
  9. dApp (Decentralized Application): An application that runs on a blockchain or peer-to-peer network instead of a single centralized server.
  10. Smart Contract: Self-executing code stored on a blockchain that automatically enforces rules and agreements.

How These Concepts Connect to the Quiz

The quiz on this site asks about the differences between Web 1.0, Web 2.0, and Web 3.0. Understanding key ideas like decentralization, dApps, and smart contracts will help you answer the Web 3.0 questions correctly.

For example, Web 1.0 is mostly about static pages, Web 2.0 adds user-generated content and interaction, and Web 3.0 builds on these with protocols and technologies that give users more control over data and value.
